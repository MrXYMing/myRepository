#### 1. 输入法乱码的解决情况
1. 设置 - 时间和语言；
2. 区域和语言 - ‘语言’一栏中‘添加语言’下选中当前语言 - 选项；
3. ‘键盘’一栏中‘微软拼音’ - 选项 - 常规；
4. ‘默认模式’下‘选择输出简体或繁体’

#### 2. mysql查询表结构改变

show table status from `smodb3.0`

#### 3. 键盘快捷键操作浏览器
- F1：打开一个新标签页；
- F3：页面查找 == ctrl+F；
- F6：选中导航栏URL；

#### 4. 进入开机启动项

1. 点击“开始”菜单，在最下面的程序搜索中输入：msconfig ，然后按回车键；
2. 在弹出的窗口里，点击“启动”选项卡，就能看到当前一些启动的程序。
